## 语法
- 语法：`@media media-type operator (media-feature) {...}`
- 参数：
	- `media-type`
		- `all`：所有设备（默认值）
		- `print`：打印预览
		- `screen`：屏幕
	- `operator`
		- `and`：与
		- `or`：或
		- `,`：或
		- `not`：非
		- `only`：查询匹配时才应用样式
	- `media-feature`
		- `aspect-ratio`：视口宽高比
		- `height`：视口高度
		- `width`：视口宽度
		- `prefers-color-scheme`：用户偏爱配色

`min-width: 10em`：等价于 `width >= 10em`
`max-width: 10em`：等价于 `width < 10em`
## 例子
1. 在某个宽度范围应用样式
```css
@media (40em <= width <= 50em) {
	.box {
		background-color: red;
	}
}
```
2. 夜间模式
```css
@media (prefers-color-scheme: dark) {
	.box {
		background-color: black;
	}
}
```
